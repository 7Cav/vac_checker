# vac_checker

XenForo addon (`Cav7/SteamChecker`) that checks Steam accounts for VAC bans. Addon code lives under `src/addons/Cav7/SteamChecker/`.

## Agent skills

### Issue tracker

Issues are tracked on GitHub (`7Cav/vac_checker`) via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage label strings are used as-is (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
